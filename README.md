# 🎓 Student Management System (C++)

## 📌 Overview

The **Student Management System** is a console-based application written in C++ that allows users to manage student records efficiently.
It supports operations like adding, displaying, searching, updating, and deleting student data.

This project is ideal for beginners to understand **Object-Oriented Programming (OOP)** concepts in C++.

---

## 🚀 Features

* ➕ Add new student records
* 📋 Display all students
* 🔍 Search student by ID
* ✏️ Update student details
* ❌ Delete student record
* 🧠 Simple and user-friendly interface

---

## 🛠️ Technologies Used

* Language: **C++**
* Concepts:

  * Classes & Objects
  * Vectors (STL)
  * Loops & Conditionals
  * Functions
